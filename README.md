# Tribute Website – Dr. A. P. J. Abdul Kalam

## Project Overview
This project is a simple Tribute Website dedicated to Dr. A. P. J. Abdul Kalam, the former President of India and renowned aerospace scientist. 
The website provides information about his life, achievements, and biographies using a clean and responsive design.
The project is built using HTML and CSS.

# Technologies Used
- HTML5
- CSS3

  Possible upgrades:
